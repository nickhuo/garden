# Research Digests

Daily filtered digest of HuggingFace Daily Papers, tailored to Nick's AI Engineer job search + Donut Labs agent work.

## Files

| File | Purpose | Who edits |
|---|---|---|
| `screening-criteria.md` | Living rubric: what to keep / drop, output format, tag legend | Nick + Digital Twin (ask the Twin to `update screening criteria: ...`) |
| `fetch_papers.py` | Pure data fetcher — hits HF index + per-paper API, emits JSON | Edit only if HF schema changes |
| `{YYYY-MM-DD}.md` | One digest per day | Generated by scheduled task |
| `README.md` | This file | Occasionally |

## How the daily pipeline works

1. Scheduled task fires at **09:00 America/Chicago** every day.
2. Task computes today's date, reads `screening-criteria.md`, runs `fetch_papers.py {DATE}`.
3. For each paper, the Digital Twin applies the criteria with judgment (not just keyword match) and distills to Problem / Method / Key result.
4. PDF links embedded as `[{id}](https://arxiv.org/pdf/{id})`.
5. Output saved to `{DATE}.md`.

## Per-paper format

```
### 🟢 [arxiv_id] Short title — [PDF](https://arxiv.org/pdf/arxiv_id) · [HF](https://huggingface.co/papers/arxiv_id)
- **Problem:** ≤20 words
- **Method:** ≤25 words, name the technique
- **Key result:** ≤20 words, quote metrics from abstract or say "No quantitative result reported"
- *repo:* {github link if exists} · ⭐{stars}   (only if GH repo present)
```

## Tag legend

- 🟢 **Directly actionable** — read / tweet / blog candidate
- 🟡 **Adjacent** — useful context, memory-worthy
- ⚪ **Watch** — field-of-view only

## Evolving the criteria

When Nick's focus shifts (new project, new technical interest, new job target):

```
> Twin, update screening criteria: add "post-training for reasoning models" to Core, drop "voice/multimodal agents"
```

The Twin edits `screening-criteria.md`, bumps the `version` in frontmatter, and appends a row to the update log.

## Running on demand

```bash
# Generate for an arbitrary date
python3 fetch_papers.py 2026-04-22 > /tmp/papers.json
```

Or just ask the Twin: "generate today's research digest" — it will execute the same pipeline as the scheduled task.
