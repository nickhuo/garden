## Part 2 — Research Radar (HuggingFace Papers, English)

### TL;DR — Today's Research Wind

Agentic search dominated today — both OpenSearch-VL and RTriever attack the same bottleneck from opposite ends: one trains the agent's search-and-reason loop via multi-turn RL, the other upgrades the retriever underneath to supply better evidence portfolios. Meanwhile, ResRL's SVD-based gradient surgery for RLVR diversity is a clean post-training trick that transfers broadly, including to agent and function-calling tasks.

### Agent frameworks & self-evolution

#### 🟢 [2605.05185] OpenSearch-VL: Open Recipe for Frontier Multimodal Search Agents
- **Problem:** Top multimodal search agents are closed-source with no reproducible training recipe.
- **Method:** Agentic RL (multi-turn fatal-aware GRPO) over a unified tool env (text/image search, OCR, cropping, super-res) with curated SFT+RL data.
- **Key result:** 10+ point avg improvement across 7 benchmarks, matching proprietary models.
- *Links:* [PDF](https://arxiv.org/pdf/2605.05185) · [HF](https://huggingface.co/papers/2605.05185)

### Agent reliability, evaluation & safety

#### 🟡 [2604.20441] MedSkillAudit: Domain-Specific Audit for Agent Skills
- **Problem:** Agent skills deployed as modular units lack pre-deployment quality auditing for domain safety.
- **Method:** Layered audit framework scoring 75 medical research agent skills on integrity, validity, reproducibility, and boundary safety.
- **Key result:** System-consensus score divergence (SD=9.5) smaller than inter-expert divergence (SD=12.4).
- *Links:* [PDF](https://arxiv.org/pdf/2604.20441) · [HF](https://huggingface.co/papers/2604.20441)

### Post-training (RL, distillation, merging, calibration)

#### 🟢 [2605.00380] ResRL: Negative Sample Projection Residual RL for LLM Reasoning
- **Problem:** RLVR over-incentivizes positive rewards, killing generation diversity in LLM reasoning.
- **Method:** SVD-based projection of negative-token hidden states onto positive subspace; residuals modulate negative gradients to decouple shared semantics.
- **Key result:** +9.4% Avg@16 and +7.0% Pass@128 over NSR on math reasoning; gains on agent/function-calling tasks.
- *Links:* [PDF](https://arxiv.org/pdf/2605.00380) · [HF](https://huggingface.co/papers/2605.00380) · [GitHub](https://github.com/1229095296/ResRL)

#### 🟡 [2605.03314] SxS: Learning Disclosure Policies for LLM Reasoning
- **Problem:** Single-stream autoregressive LLMs face a silence tax — more thinking delays first output, early streaming risks premature commitment.
- **Method:** Side-by-Side interleaved reasoning trained with SFT (entailment-aligned trajectories) then RL, controlling when to disclose vs. continue private reasoning.
- **Key result:** Improved accuracy-latency Pareto on AIME25 and GPQA-Diamond across Qwen3-30B-A3B and Qwen3-4B.
- *Links:* [PDF](https://arxiv.org/pdf/2605.03314) · [HF](https://huggingface.co/papers/2605.03314)

### Retrieval / RAG / memory

#### 🟢 [2605.04018] RTriever: Reasoning-Intensive Retrieval for Agentic Search
- **Problem:** Existing retrieval benchmarks and training optimize single-passage relevance, not evidence portfolios for multi-step agent reasoning.
- **Method:** BRIGHT-Pro benchmark with multi-aspect gold evidence + agentic search protocol; RTriever-4B via LoRA on Qwen3-Embedding-4B with aspect-decomposed synthetic data.
- **Key result:** Aspect-aware + agentic eval exposes hidden retriever failures; RTriever-4B substantially outperforms base model.
- *Links:* [PDF](https://arxiv.org/pdf/2605.04018) · [HF](https://huggingface.co/papers/2605.04018)
