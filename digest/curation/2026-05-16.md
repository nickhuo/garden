## Part 2 — Research Radar (HuggingFace Papers, English)

### TL;DR — Today's Research Wind
Agent memory is the dominant theme — three independent papers tackle different failure modes: stale facts (STALE), visual grounding gaps (MemEye/MemLens), and cold-start without prior tasks (PREPING). The second clear thread is self-improving agents via RL: Self-Distilled Agentic RL and EvolveMem both push toward agents that improve their own training signal. Together this cluster is directly applicable to the kind of long-session, multi-tool agents Nick is building at Donut Labs.

### Agent Frameworks & Self-Evolution

#### 🟢 [2605.14892] Beyond Individual Intelligence: Multi-Agent Collaboration, Failure Attribution, Self-Evolution Survey
- **Problem:** Multi-agent LLM systems lack systematic understanding of coordination failures and self-improvement mechanisms.
- **Method:** Structured survey across collaboration protocols, failure attribution methods, and self-evolution architectures in LLM-based MAS.
- **Key result:** No quantitative result reported (survey paper); identifies open problems in scalable coordination and failure isolation.
- *Links:* [PDF](https://arxiv.org/pdf/2605.14892) · [HF](https://huggingface.co/papers/2605.14892)

#### 🟢 [2605.15040] Orchard: An Open-Source Agentic Modeling Framework
- **Problem:** Existing agentic frameworks lack unified treatment of planning, tool use, and multi-turn environment interaction.
- **Method:** Orchard provides a modular open-source framework for building and evaluating LLM agents across planning, reasoning, and tool-use tasks.
- **Key result:** No quantitative result reported; open-source release.
- *Links:* [PDF](https://arxiv.org/pdf/2605.15040) · [HF](https://huggingface.co/papers/2605.15040)

#### 🟢 [2605.14051] SPIN: Structural LLM Planning via Iterative Navigation for Industrial Tasks
- **Problem:** LLM planners in industrial agent pipelines frequently generate structurally invalid or overly long workflows.
- **Method:** SPIN uses iterative navigation with structural constraints to guide LLM plan generation toward valid, compact outputs.
- **Key result:** No quantitative result reported.
- *Links:* [PDF](https://arxiv.org/pdf/2605.14051) · [HF](https://huggingface.co/papers/2605.14051)

#### 🟡 [2605.15198] ATLAS: Agentic or Latent Visual Reasoning? One Word is Enough for Both
- **Problem:** Visual reasoning requiring intermediate states forces a choice between generating full images (slow) or purely latent computation (weak).
- **Method:** ATLAS uses a single token to switch between agentic (image-generating) and latent (hidden-state) reasoning modes.
- **Key result:** No quantitative result reported.
- *Links:* [PDF](https://arxiv.org/pdf/2605.15198) · [HF](https://huggingface.co/papers/2605.15198)

### Agent Reliability, Evaluation & Safety

#### 🟢 [2605.06527] STALE: Can LLM Agents Know When Their Memories Are No Longer Valid?
- **Problem:** Agent memory benchmarks test static fact retrieval; they ignore whether agents detect that stored facts have become stale.
- **Method:** STALE benchmark with time-varying fact updates; tests agents' ability to flag outdated memory vs. retrieve current information.
- **Key result:** Current agents fail significantly at staleness detection; retrieval accuracy degrades sharply as memory ages.
- *Links:* [PDF](https://arxiv.org/pdf/2605.06527) · [HF](https://huggingface.co/papers/2605.06527)

#### 🟢 [2605.10912] WildClawBench: Real-World, Long-Horizon Agent Evaluation
- **Problem:** Most agent benchmarks use synthetic tasks; real CLI-harness agent behavior in the wild remains under-measured.
- **Method:** WildClawBench: CLI-based benchmark with long-horizon, real-world tasks collected from actual user sessions.
- **Key result:** No quantitative result reported; gap between synthetic and real-world agent performance highlighted.
- *Links:* [PDF](https://arxiv.org/pdf/2605.10912) · [HF](https://huggingface.co/papers/2605.10912)

#### 🟢 [2605.14906] MemLens: Benchmarking Multimodal Long-Term Memory in Large Vision-Language Models
- **Problem:** Long-term LVLM memory evaluation rarely tests preservation of visual evidence needed for later reasoning.
- **Method:** MemLens benchmark across long-context LVLMs and memory-augmented agents; measures visual grounding retention.
- **Key result:** Memory-augmented agents retain visual evidence significantly better than long-context-only approaches on MemLens tasks.
- *Links:* [PDF](https://arxiv.org/pdf/2605.14906) · [HF](https://huggingface.co/papers/2605.14906)

#### 🟢 [2605.15128] MemEye: A Visual-Centric Evaluation Framework for Multimodal Agent Memory
- **Problem:** Agent memory evals use text-only metrics; visual grounding gaps go undetected.
- **Method:** MemEye framework evaluates whether agents preserve visual evidence across multi-step interactions.
- **Key result:** No quantitative result reported; framework release.
- *Links:* [PDF](https://arxiv.org/pdf/2605.15128) · [HF](https://huggingface.co/papers/2605.15128)

### Post-Training (RL, Distillation, Merging)

#### 🟢 [2605.15155] Self-Distilled Agentic Reinforcement Learning
- **Problem:** RL for long-horizon agents provides only coarse trajectory-level reward; step-level supervision is missing.
- **Method:** Self-distillation: agent generates its own dense supervision signal from trajectory rollouts; combined with RL fine-tuning.
- **Key result:** Outperforms on-policy RL baselines on long-horizon agentic tasks; no specific metric reported in abstract.
- *Links:* [PDF](https://arxiv.org/pdf/2605.15155) · [HF](https://huggingface.co/papers/2605.15155)

#### 🟡 [2605.13301] Achieving Gold-Medal-Level Olympiad Reasoning via Simple and Unified Scaling
- **Problem:** Frontier reasoning models hit gold-medal performance on math olympiads; key scaling factors remain unclear.
- **Method:** Simple unified scaling approach across compute, data, and model capacity; evaluated on IMO-level benchmarks.
- **Key result:** Gold-medal-level performance on international math olympiad benchmarks.
- *Links:* [PDF](https://arxiv.org/pdf/2605.13301) · [HF](https://huggingface.co/papers/2605.13301)

#### 🟡 [2605.14386] Darwin Family: Training-Free Evolutionary Merging for LLM Reasoning
- **Problem:** Frontier reasoning requires expensive training; model merging offers a training-free alternative if done correctly.
- **Method:** Gradient-free weight-space recombination with MRI-trust weighting; evolutionary search over merge configurations.
- **Key result:** Achieves frontier-level reasoning performance without additional training on several benchmarks.
- *Links:* [PDF](https://arxiv.org/pdf/2605.14386) · [HF](https://huggingface.co/papers/2605.14386)

#### 🟡 [2605.11458] Adaptive Teacher Exposure for Self-Distillation in LLM Reasoning
- **Problem:** On-policy self-distillation uses fixed teacher exposure schedule; suboptimal supervision during training.
- **Method:** Adaptive scheduling of teacher conditioning during rollouts based on student performance signals.
- **Key result:** Improved reasoning benchmark scores vs. fixed-schedule baselines; specific deltas not in abstract.
- *Links:* [PDF](https://arxiv.org/pdf/2605.11458) · [HF](https://huggingface.co/papers/2605.11458)

### Retrieval / RAG / Memory

#### 🟢 [2605.13941] EvolveMem: Self-Evolving Memory Architecture via AutoResearch for LLM Agents
- **Problem:** Agent memory systems treat retrieval infrastructure as fixed; stored content evolves but scoring functions do not.
- **Method:** EvolveMem automatically researches and updates both the stored content and the retrieval scoring functions across sessions.
- **Key result:** No quantitative result reported; self-evolving memory outperforms static systems on multi-session agent tasks (qualitative).
- *Links:* [PDF](https://arxiv.org/pdf/2605.13941) · [HF](https://huggingface.co/papers/2605.13941)

#### 🟢 [2605.13880] PREPING: Building Agent Memory without Tasks
- **Problem:** Agent memory systems face a cold-start gap: useful memory requires prior task interactions, but agents need memory before first deployment.
- **Method:** PREPING constructs memory proactively from unstructured knowledge (no task demonstrations required) using structured pre-population.
- **Key result:** Agents with PREPING memory outperform no-memory baselines from first interaction; exact metrics not in abstract.
- *Links:* [PDF](https://arxiv.org/pdf/2605.13880) · [HF](https://huggingface.co/papers/2605.13880)

### Inference Optimization & Architecture

#### 🟡 [2605.00180] RouteProfile: Elucidating the Design Space of LLM Profiles for Routing
- **Problem:** LLM routing lacks principled understanding of what profile features drive routing decisions across diverse queries.
- **Method:** Systematic design-space study of LLM profile representations for routing; evaluates profile types against query distributions.
- **Key result:** Profile design choices have large impact on routing accuracy; optimal profile type varies by query domain.
- *Links:* [PDF](https://arxiv.org/pdf/2605.00180) · [HF](https://huggingface.co/papers/2605.00180)

#### 🟡 [2605.06554] Long Context Pre-Training with Lighthouse Attention
- **Problem:** Quadratic SDPA memory cost bottlenecks training causal transformers at extreme sequence lengths.
- **Method:** Lighthouse Attention: training-time approximation that replaces full SDPA with a sparse attention pattern for long contexts.
- **Key result:** No quantitative result reported; enables pre-training at sequence lengths impractical with standard SDPA.
- *Links:* [PDF](https://arxiv.org/pdf/2605.06554) · [HF](https://huggingface.co/papers/2605.06554)

### Code / Web Agents

#### 🟢 [2605.14445] FrontierSmith: Synthesizing Open-Ended Coding Problems at Scale
- **Problem:** LLM coding benchmarks focus on well-defined tasks; open-ended real-world coding challenges with no known optimal solution are underrepresented.
- **Method:** FrontierSmith: automated pipeline for synthesizing diverse, open-ended coding problems at scale using LLMs.
- **Key result:** Synthesized problems expose capability gaps not visible in standard benchmarks; specific metrics not in abstract.
