## Part 2 — Research Radar（HuggingFace Papers）

### TL;DR — 本日 research 风向
今天的主导主题是 **agent 的"评估与自我进化"**：从把 skill 当作可测试长生命周期资产（MUSE-Autoskill），到对 multi-agent 工作流做轨迹级幻觉审计（Trajel）和多层级自动评估（Agentic CLEAR），整条线都在回答"如何让 agent 可信、可观测、可持续改进"——这正好压在你 Donut Labs 的 46 workflows narrative 上。第二条线是 **agent-native 的训练与基础设施**：MiniMax-M2 把 mini-activation MoE 端到端做成 agentic 模型，MobileGym 给 GUI agent 造可验证的并行 RL 环境，Gemini Embedding 2 为 agentic RAG 提供原生多模态检索底座。

### Agent frameworks & self-evolution

### 🟢 [2605.27366] MUSE-Autoskill：以 skill 生命周期驱动的自进化 agent
- **Problem:** 现有 skill 创建把 skill 当孤立静态产物，复用性、可靠性、长期改进都受限。
- **Method:** skill-centric 框架，统一生命周期（创建/记忆/管理/评估/精炼）+ skill 级记忆累积跨任务经验，用单测与运行反馈持续 refine。
- **Key result:** SkillsBench 51 任务带人写 skill 达 68.40%（+15.21pp）；自生成 skill 在 35 任务达 87.94%，超人写上限；跨 agent 迁移 +10.51pp。
- *Links:* [PDF](https://arxiv.org/pdf/2605.27366) · [HF](https://huggingface.co/papers/2605.27366)

### 🟡 [2605.26494] The MiniMax-M2 Series：mini activation 释放最大现实智能
- **Problem:** 如何在低激活成本下做到前沿级 agentic 性能并支持长程 agent 轨迹训练。
- **Method:** 229.9B 总参 / 仅 9.8B 激活的 MoE；agent 驱动可验证轨迹数据 + Forge agent-native RL 系统；M2.7 checkpoint 已能自调试训练、改自己的 scaffold（自进化雏形）。
- **Key result:** M2.7 SWE-bench Pro 56.2、Terminal-Bench 2.0 57.0、BrowseComp 77.8、AIME 2026 94.2，仅 4.3% 激活比。
- *Links:* [PDF](https://arxiv.org/pdf/2605.26494) · [HF](https://huggingface.co/papers/2605.26494)

### 🟡 [2605.24468] SAM：长程推理 agent 的状态自适应记忆
- **Problem:** 长程任务中 agent 记忆管理难以随状态变化自适应。
- **Method:** State-Adaptive Memory 机制，按推理状态动态调度记忆内容。
- **Key result:** 摘要未取（仅元数据），未抓到量化 metric。
- *Links:* [PDF](https://arxiv.org/pdf/2605.24468) · [HF](https://huggingface.co/papers/2605.24468)

### Agent reliability, evaluation & safety

### 🟢 [2605.24219] Beyond Final Answers：multi-agent 工业工作流的轨迹级幻觉审计
- **Problem:** 多数幻觉 benchmark 只看最终输出，漏掉中间 Thought-Action-Observation 步骤里产生的失败。
- **Method:** Trajel 数据集 + 五类幻觉分类法（事实/指代/逻辑/流程/范围），在子任务、轨迹、长上下文三级 benchmark 监督检测模型（基于 AssetOpsBench 专家标注 trace）。
- **Key result:** agent loop 中的轻量执行质量信号（clarity-and-justification，AUC=0.908）比监督轨迹分类器（best AUC=0.689）更能预测幻觉；近半幻觉轨迹同时含多类型。
- *Links:* [PDF](https://arxiv.org/pdf/2605.24219) · [HF](https://huggingface.co/papers/2605.24219)

### 🟢 [2605.22608] Agentic CLEAR：LLM agent 的自动多层级评估
- **Problem:** 现有工具多停留在 observability + 静态手写错误分类，无法适配新领域。
- **Method:** 凌驾于 observability 层之上的自动、动态评估框架，输出系统/trace/node 三粒度文本洞察，带直观 UI。
- **Key result:** 4 benchmark、7 种 agentic 设置、数万次 LLM 调用上，反馈与人工标注错误强对齐，并能预测任务成功率（IBM Research，[代码](https://ibm.biz/ACLEAR-Code)）。
- *Links:* [PDF](https://arxiv.org/pdf/2605.22608) · [HF](https://huggingface.co/papers/2605.22608)

### 🟡 [2605.27141] VitaBench 2.0：长期交互中的个性化与主动性 agent 评估
- **Problem:** 现有 agent benchmark 只测推理与工具调用，忽略从碎片化日常交互推断并利用用户偏好。
- **Method:** 按用户时间序组织任务，偏好嵌在异构交互中；提供可扩展记忆接口做受控对比；并评估"主动获取缺失信息"能力。
- **Key result:** 最佳模型 Claude-Opus-4.6(thinking) Avg@4 仅 0.503、Pass^4 0.337；所有模型 proactive 分数显著低于 personalization（如 Claude 27.4 vs 46.0）。[代码](https://github.com/meituan-longcat/vitabench-2.0)
- *Links:* [PDF](https://arxiv.org/pdf/2605.27141) · [HF](https://huggingface.co/papers/2605.27141)

### 🟡 [2605.25188] DarkForest：少说话、更高准确率的 multi-agent LLM

### Retrieval / RAG / memory

### 🟢 [2605.27295] Gemini Embedding 2：来自 Gemini 的原生多模态 embedding
- **Problem:** 现有多模态 embedding（CLIP/SigLIP 等晚融合）难处理混合模态输入，跨模态语义交互弱。
- **Method:** 从 Gemini 初始化的双向注意力 encoder，多任务多阶段对比学习（NCE + in-batch negatives），原生处理 text/image/audio/video 任意交错输入，MRL 支持 768/1536/3072 维。
- **Key result:** MSCOCO 62.9 R@1、Vatex 68.8 NDCG@10、MTEB multilingual 69.9、MTEB Code 84.0；原生 audio 检索 mrr@10 73.99 优于 ASR 级联 70.40。直指 agentic RAG。
- *Links:* [PDF](https://arxiv.org/pdf/2605.27295) · [HF](https://huggingface.co/papers/2605.27295)

### Code / web agents

### 🟡 [2605.26114] MobileGym：可验证、高并行的移动 GUI agent 仿真平台
- **Problem:** GUI agent 缺少可验证 outcome 信号与低成本并行 RL 环境。
- **Method:** 浏览器托管、用结构化 JSON state 做确定性判定；单服务器托管 ~400 并行实例（各 ~400MB / ~3s 冷启动）；MobileGym-Bench 提供 416 个参数化任务模板覆盖 28 个 app。
- **Key result:** Qwen3-VL-4B 上 GRPO 在 256 任务测试集 +12.8pt（9.4%→22.2%）；真机执行保留 95.1% 仿真训练增益。
- *Links:* [PDF](https://arxiv.org/pdf/2605.26114) · [HF](https://huggingface.co/papers/2605.26114)
