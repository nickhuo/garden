### Agent 可靠性、评估与记忆 (Agent reliability, evaluation & memory)

#### 🟢 [2605.28732] MemTrace — LLM 记忆系统的错误溯源
- **Problem:** LLM agent 的 memory layer 出错时无法定位是 retrieval / write / consolidation 哪一环引入了错误。
- **Method:** 在每条 memory 记录上挂 provenance trace，错误发生时反向归因到具体记忆事件。
- **Key result:** No quantitative result reported in summary.
- *Links:* [PDF](https://arxiv.org/pdf/2605.28732) · [HF](https://huggingface.co/papers/2605.28732)

#### 🟢 [2605.28775] Learn from Weaknesses — 小型 computer-use agent 的自动领域专精
- **Problem:** 通用 small computer-use agent 在专门领域（如某个具体网站、某类 IDE）能力差，人工微调成本高。
- **Method:** 通过失败案例自动诊断弱点维度，针对性合成训练数据做领域专精。
- **Key result:** No quantitative result reported in summary.
- *Links:* [PDF](https://arxiv.org/pdf/2605.28775) · [HF](https://huggingface.co/papers/2605.28775)

### Self-improvement / Post-training

#### 🟡 [2605.28814] Self-Improving LMs with Bidirectional Evolutionary Search (Harvard)
- **Problem:** 现有 self-improvement 多是单向（生成→筛选），陷入局部最优。
- **Method:** 双向演化搜索——同时从 task 端和 solution 端做变异，借鉴遗传算法的双向 crossover。
- **Key result:** No quantitative result reported in summary.
- *Links:* [PDF](https://arxiv.org/pdf/2605.28814) · [HF](https://huggingface.co/papers/2605.28814)

#### 🟡 [2605.28421] DenoiseRL — 从带噪 prefix 中恢复推理
- **Problem:** Reasoning 模型对 prompt 前缀里的 typos / 错误事实极敏感，整条 CoT 都会被带偏。
- **Method:** RL bootstrapping，让模型显式学会"先识别再纠正"噪声前缀。
- **Key result:** No quantitative result reported in summary.
- *Links:* [PDF](https://arxiv.org/pdf/2605.28421) · [HF](https://huggingface.co/papers/2605.28421)
