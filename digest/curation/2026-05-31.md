## Part 2 — Research Radar（HuggingFace Papers）

### TL;DR — 本日 research 风向
今天 60 篇里主导主题是 **agent 的"环境"与"记忆"**：怎么合成可验证的训练环境（LiteCoder-Terminal、PhoneWorld）、怎么在 cloud/device 之间切（Hybrid Multi-Agent）、怎么把 skill 既外置又内化（Skill0.5）。第二条线是 **长程推理的工程成本**——CONF-KV 用模型自身的 confidence 来管 KV cache 驱逐。安全侧 AgentDoG 1.5 把 guardrail 压到 1B 级、1k 样本。检索方向偏"机理解释"（Xetrieval、position bias）而非新 SOTA。

### Agent frameworks & self-evolution

### 🟢 [2605.28424] Skill0.5: Joint Skill Internalization and Utilization
- **Problem:** skill-based RL 被迫二选一——全外置（上下文开销）或全内化（过拟合）。
- **Method:** 动态 router，对通用 skill 与任务专用 skill 分别处理，联合内化+使用。
- **Key result:** 在 ALFWorld / WebShop 上同时超过 memory-based 与 skill-based 基线（in/out-of-distribution 均提升）；abstract 未给具体数字。
- *Links:* [PDF](https://arxiv.org/pdf/2605.28424) · [HF](https://huggingface.co/papers/2605.28424)

### 🟢 [2605.30102] When Cloud Agents Meet Device Agents: Hybrid Multi-Agent Systems
- **Problem:** cloud LLM + on-device SLM 的混合 agent 缺系统化设计原则，现状都是 ad hoc。
- **Method:** 把两种代表性多 agent 架构改造成支持混合推理，实测 power-cost-performance Pareto 前沿。
- **Key result:** 最优架构高度任务相关；更多前沿算力不必然带来更好性能。No quantitative result reported（定性结论）。
- *Links:* [PDF](https://arxiv.org/pdf/2605.30102) · [HF](https://huggingface.co/papers/2605.30102)
