# Introducing Contextual Retrieval

**Source:** Anthropic Engineering Blog, September 19 2024
**URL:** https://www.anthropic.com/engineering/contextual-retrieval

## Summary

Standard RAG pipelines chunk documents then embed those chunks in isolation, discarding surrounding context. Anthropic introduces [[Contextual Retrieval]]: before embedding, each chunk is augmented with a short AI-generated blurb that situates it within the source document. This context is generated by Claude with prompt caching (caching the full document, varying only the chunk), making it cheap enough for production.

The technique is composed with two further enhancements:
- **[[BM25 and Hybrid Retrieval]]**: combining dense embedding scores and sparse BM25 keyword scores via Reciprocal Rank Fusion (RRF) captures both semantic and lexical signals.
- **[[Reranking]]**: a cross-encoder reranker (e.g., Cohere Rerank) re-scores the top-k hybrid candidates for direct query relevance.

## Key Claims

1. Context-stripping is the primary failure mode in standard RAG. Chunks like "revenue declined 3% in Q3" become uninterpretable without document-level anchor.
2. Contextual embedding alone: top-20 recall ~85%, reducing failed retrievals ~49% vs baseline (~77%).
3. Hybrid (embed + BM25) without context: ~83% recall, ~35% failure reduction.
4. Contextual + Hybrid: ~91% recall, ~57% failure reduction.
5. Contextual + Hybrid + Rerank: ~95%+ recall, **~67% failure reduction** — best configuration.
6. Prompt caching cuts per-chunk context generation cost ~75%, making it economically viable at scale.

## Evaluation Setup

Internal benchmark: 50 documents, 5000 questions. Metric: % of correct chunks appearing in top-20 retrieved results. Tested embedding-only, BM25-only, hybrid, and all contextual variants.

## Practical Tradeoff Ladder

| Tier | Method | Notes |
|---|---|---|
| 1 | Embedding-only | Cheapest |
| 2 | Embedding + BM25 (hybrid) | No added LLM cost |
| 3 | Contextual Retrieval + Hybrid | Needs Claude + caching |
| 4 | + Reranking | Highest precision, highest cost |
